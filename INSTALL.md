# Installation Protocol

### Initialisation Components
The Strata package consists of three core architectural files:

- **manifest.json** - Plugin metadata for Figma orchestration.
- **code.js** - The logic gateway for design resolution.
- **ui.html** - The interface for intent declaration.

### Execution Steps
1. **Acquire:** Download the latest release archive from the `repository/` directory.  

2. **Import:** In the Figma App, navigate to **Plugins → Development → Import plugin from manifest…**  

3. **Resolve:** Select the `manifest.json` file from the extracted package.  

4. **Deploy:** Strata will now be available within your Figma environment under the Development menu.  

### Evaluation Phase
Upon first execution and account creation, your profile will be granted **5 Resolution Credits** for the first 30 days. This allows for full-fidelity verification of the engine's structural output prior to subscription activation.

---

### Repository Policy
The `repository/` directory serves as the official distribution point for the Strata resolution engine. All updates are delivered as version-controlled `.zip` archives.
